# Tuần 01: Khởi tạo dự án & Thiết lập môi trường (08/06 - 14/06/2026)

### Khảo sát và xác định yêu cầu:
Phân tích các nền tảng chia sẻ công thức nấu ăn hiện có (Cookpad, Yummly), thống nhất phạm vi chức năng cốt lõi cho MealHub (Đăng nhập, CRUD công thức, Tìm kiếm).

### Thiết lập hạ tầng kỹ thuật:
Tạo Repository GitHub với tên ASPNET-da21tta-nguyenngocduyen-MealHub, mời giảng viên làm Collaborator. Cài đặt môi trường Visual Studio 2022, .NET 7/8 SDK và SQL Server.

### Nghiên cứu công nghệ:
Tìm hiểu cấu trúc ASP.NET Core MVC, kiến trúc Razor View và cơ chế Routing.

### Kết quả đạt được:
Hoàn thành tài liệu đặc tả yêu cầu sơ bộ. Dự án đã được khởi tạo và có thể build thành công trên máy local. File README.md đã được cập nhật.
