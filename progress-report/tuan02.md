# Tuần 02: Xây dựng Core & Database (15/06 - 21/06/2026)

### Thiết kế CSDL:
Xây dựng sơ đồ ERD với các bảng Categories, Recipes (tích hợp Cloudinary) và AspNetUsers. Thiết lập quan hệ khóa ngoại.

### Phát triển tầng Data:
Triển khai ApplicationDbContext (Code-First), tạo các Model Entity và chạy Migration để tạo Database.

### Xây dựng Repository:
Viết Interface và Class RecipeRepository, triển khai các phương thức CRUD cơ bản kết hợp với LINQ.

### Kết quả đạt được:
Database đã được tạo thành công. Hoàn thành tầng truy xuất dữ liệu (Repository) và tiến hành test thử với Postman/Swagger.
