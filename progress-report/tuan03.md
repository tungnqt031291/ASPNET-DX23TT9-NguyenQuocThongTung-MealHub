# Tuần 03: Hiện thực Logic nghiệp vụ & Service (22/06 - 28/06/2026)

### Phát triển Service Layer:
Hoàn thiện RecipeService để xử lý logic nghiệp vụ như tăng view count, quản lý người dùng. Tích hợp ASP.NET Core Identity để quản lý xác thực.

### Xây dựng giao diện Layout:
Tạo _Layout.cshtml base, tích hợp Bootstrap 5 cho responsive. Hoàn thiện giao diện Header, Footer và thanh điều hướng.

### Tích hợp TinyMCE:
Cài đặt trình soạn thảo văn bản cho trường Instructions để người dùng nhập hướng dẫn nấu với định dạng phong phú.

### Kết quả đạt được:
Kiến trúc 3 lớp đã được thiết lập (Controller-Service-Repository). Giao diện khung của trang web đã sẵn sàng.
